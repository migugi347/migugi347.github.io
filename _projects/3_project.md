# KANM Student Radio Website

I worked with a student team to help build the website for KANM, Texas A&M’s student radio station. The project focused on creating a stronger web presence for the organization with a UI that felt more polished, modern, and easier to navigate.

## Contributions

- Built frontend components in React
- Collaborated with a team of about 10 students across design and implementation
- Used Figma to shape the interface before development
- Helped deliver a public-facing site for a real campus organization

## Takeaway

This project was a good reminder that strong web work is not just about code. It also means translating ideas into usable layouts, collaborating with stakeholders, and shipping something that represents a group well online.
